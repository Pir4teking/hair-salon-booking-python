# Hair Salon Booking App

A desktop application built with Python and Tkinter for managing hair salon appointments.

## Features
- Date selection using a calendar interface.
- Name and time slot scheduling.
- User-friendly booking system.

## Technologies
- Python 3.x
- Tkinter (GUI)
- tkcalendar

## How to run
1. Clone this repository.
2. Install dependencies:
   `pip install -r requirements.txt`
3. Run the application:
   `python peluqueria.py`
